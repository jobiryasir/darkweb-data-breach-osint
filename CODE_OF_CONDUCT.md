# Code of Conduct

## Our Commitment
As a researcher in the fields of Cyber Security, OSINT, and Digital Forensics, I am committed to maintaining a professional, ethical, and collaborative environment. This repository is dedicated to educational purposes, security research, and the advancement of digital network intelligence.

## Expected Behavior
By accessing, using, or contributing to this repository, you agree to:
*   **Ethical Research:** Conduct all activities with the highest ethical standards.
*   **Legal Compliance:** Ensure all actions comply with relevant local and international laws.
*   **Professionalism:** Maintain a respectful and professional tone in all communications and contributions.
*   **Data Integrity:** Respect the privacy of individuals and refrain from using or distributing private, sensitive, or PII (Personally Identifiable Information) data.

## Prohibited Behavior
The following actions are strictly prohibited:
*   **Malicious Use:** Using any information or tools from this repository for illegal activities, cyberattacks, or unauthorized system access.
*   **Harassment:** Any form of harassment or unprofessional conduct toward contributors or users.
*   **Unverified Claims:** Sharing or promoting unverified or misleading intelligence data.

## Reporting Issues
If you encounter any content that violates this code of conduct or if you wish to report a security vulnerability regarding the repository itself, please contact me directly via the provided channels (e.g., Facebook).

## Scope
This code of conduct applies to all contributors, users, and visitors of this repository. Your participation is an acknowledgment of these standards.

*“Stay Secure. Stay Anonymous.”*
