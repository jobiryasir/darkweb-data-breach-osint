### 💡 Security Advisory
> **Warning:** Integrating Tor Browser with the I2P network can create unique browser fingerprinting risks. It is strongly recommended to use a separate dedicated browser profile (such as a standard Firefox instance) for I2P traffic, configured to use the I2P local proxy (typically port 4444) to maintain the integrity of your Tor-specific anonymity.
