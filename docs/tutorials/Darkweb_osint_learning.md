### 🛡️ Search OpSec Parameters
- **Avoid Trailing Footprints:** When using clearnet gateways, copy link addresses manually into your hardened Tor Browser or I2P environment.
- **Query Compartmentalization:** Do not search for personal handles, locations, or unique pseudonyms inside deep-web engines; crawl engines log historical string correlations.
- **Metadata Leak Warning:** Be cautious with downloadable indexed images/attachments.
- **Information Slippage:** Monitor shared markers (usernames, PGP, crypto addresses) across clearnet/darknet.
